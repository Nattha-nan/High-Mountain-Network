# High-Mountain Resilient Network
## Project Pitch
CP352005 – Computer Networks

---

# Problem Statement

พื้นที่ภูเขาสูงมีข้อจำกัด:

- การเชื่อมต่อไม่ต่อเนื่อง
- สภาพอากาศรุนแรง
- พลังงานจำกัด
- ต้องรองรับการสื่อสารฉุกเฉิน

เครือข่ายแบบทั่วไปไม่รองรับความทนทานระดับนี้

---

# Proposed Solution

High-Mountain Resilient Network

คุณสมบัติหลัก:

- Mesh backbone ≥2-connected
- OSPF multi-path routing
- DTN overlay สำหรับ outage
- QoS prioritization
- Energy-aware operation modes

---

# Why This Matters

- รองรับการสื่อสารฉุกเฉิน
- ลด network partition
- เหมาะกับพื้นที่ห่างไกล
- สาธิตแนวคิด resilient networking
