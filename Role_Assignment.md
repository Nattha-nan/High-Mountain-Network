# Team Structure

| Role | Responsibilities |
|--------|----------------|
| Architect | Topology design, cost model, architectural validation |
| Engineer | Routing, DTN implementation, simulation |
| Specialist | Energy model, QoS, resilience metrics |
| DevOps | Repository, integration, CI, documentation |
| Tester/QA | Test plan, failure simulation, validation |

---

# Role Boundaries

## Architect
In Scope:
- System design
- Routing strategy
- Interface specification

Out of Scope:
- Detailed coding
- Writing test cases

---

## Engineer
In Scope:
- Routing implementation
- DTN buffer coding
- Simulation scripting

Out of Scope:
- Architecture redesign
- QoS policy definition

---

## Specialist
In Scope:
- QoS policy
- Energy degradation rules
- Failure modeling

Out of Scope:
- Core routing coding
- Git management

---

## DevOps
In Scope:
- Repository structure
- Branch management
- Integration coordination

Out of Scope:
- Algorithm design
- Traffic modeling

---

## Tester/QA
In Scope:
- Test design
- Failure simulation
- Performance measurement

Out of Scope:
- Architecture modification
- Implementation decisions

---
